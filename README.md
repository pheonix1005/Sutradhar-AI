# SUTRADHAR AI

**Team:** Team Yurei  
**Track:** Track A - Web Development  
**Domain Focus:** AI-Powered Smart Platforms  
**Problem Statement:** PS-A01 - Personal AI Agent Dashboard (Build Your Own Agent Workforce)

## 📌 Project Overview

**Agents AI** is the defining technology trend of 2025. Unlike traditional chatbots that simply respond to prompts, AI agents can autonomously plan, make decisions, use tools, and execute multi-step tasks on a user's behalf. 

**The Challenge:** Giving everyday users a simple, visual way to create and manage their own personal agent workforce without writing a single line of code.

Sutradhar AI aims to solve this by providing a visual, no-code dashboard where users can create, configure, and display personal AI agents.

## ⚙️ Key Requirements & MVP Features

Our Minimum Viable Product (MVP) includes the following core components:

1. **Visual Agent Builder:** An interface where users can define an agent's name, role, personality, and equip them with available tools.
2. **Tool Library:** A collection of skills agents can use (e.g., web search, summarizer, email drafter, database/API query, calculator).
3. **Multi-Agent Pipeline:** A visual workflow view where users can chain multiple agents together to complete complex, multi-step workflows.
4. **Real-time Activity Log:** An interface showing exactly what each agent is doing, thinking, and outputting step by step.
5. **Task Input Interface:** A simple command center where the user can assign tasks and watch their agents collaborate.
6. **Agent Memory Panel:** A view showing context, past conversations, and learned user preferences.

## 💼 Business Plan

- **The Problem:** Currently, AI agents and advanced automation are restricted to users who know how to code.
- **The Solution:** A visual "no-code" dashboard allowing anyone to build and manage AI agents.
- **Target Market:** Small and Medium Scale Industries (Handicraft, Tourism, and Education), specifically targeting Tier 2 and Tier 3 cities in North East India.
